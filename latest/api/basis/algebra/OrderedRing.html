OrderedRing

trait OrderedRing [OrderedRing] extends Ordered[OrderedRing] with Ring[OrderedRing]

An element of a commutative ring with a total order.

A OrderedRing has a morphism from Long values to OrderedRing value values. An Long value n represents the nth addition of the OrderedRing's multiplicative identity to its additive identity.

OrderedRing

the element type of the ordered ring.
