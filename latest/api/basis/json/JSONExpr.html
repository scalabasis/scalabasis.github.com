basis.json

JSONExpr

class JSONExpr[+Context <: Universe with Singleton, JSON <: JSONFactory with Singleton] extends JSONFactory

A JSON factory that generates expressions that build data structures for another JSON implementation.

Context

The singleton type of the reflective universe in which to generate expressions.

JSON

The singleton type of the JSON implementation to produce expressions for.

Source
JSONExpr.scala
Example: 

  1. // The expression json""" [{}, [], "", 0, 0.0, true, false, null] """
// generates the following code for the givem json implementation:
val builder = json.newBuilder(8)
builder += json.JSObject.empty
builder += json.JSArray.empty
builder += json.JSString("")
builder += json.JSInteger(0L)
builder += json.JSDecimal(0.0)
builder += json.JSTrue
builder += json.JSFalse
builder += json.JSNull
builder.result
Note

This implementation currently generates expressions non-hygienically. The typechecker will not warn you if the JSONFactory API changes. Simple reifications fail when splicing the json factory expression (Cannot materialize Expr). And I don't know how to generate the builder block expressions reifically (awesome, new word!). Suggestions welcome (send to Chris Sachs)!
