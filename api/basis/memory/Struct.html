Struct

class Struct [T] extends AnyRef

A typeclass that implements store-by-value semantics. Structs convert between fixed-length byte sequences and instances. Think of a struct as the value type of a type.

Frames

A struct's alignment and size constitute its frame. A struct's size must be a multiple of its alignment. structs must never access more than size - 1 bytes beyond any given address, and should assume that all given addresses are properly aligned.

Projections

A struct may project itself to an offset of a larger, alternately aligned frame. Projections give uniform accees to individual fields, or even groups of fields, of aggregate value types.
