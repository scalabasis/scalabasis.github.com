CompleteField

trait CompleteField [CompleteField] extends Field[CompleteField]

An element of a complete field.

A CompleteField has a morphism from Int values to CompleteField value values. An Int value n represents the nth addition of the CompleteField's multiplicative identity to its additive identity.

A CompleteField also has a morphism from Double values to CompleteField values. A Double value x represents a rational approximation to an element of this CompleteField.

CompleteField

the element type of the complete field.
